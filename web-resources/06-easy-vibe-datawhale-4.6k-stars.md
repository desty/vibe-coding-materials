## 핵심 철학

"말할 수 있으면 앱을 만들 수 있다"
프로그래밍 지식 없이도 AI 개발 도구로 아이디어를 실제 앱으로 구현.

## 대상 수강자

- **완전 초보자** — 첫 프로젝트를 만들고 싶은 분
- **PM / 창업자** — 빠른 MVP 검증, 최소 비용 개발
- **학생** — AI 시대 실용적 스킬 개발
- **주니어 개발자** — 컨셉에서 런칭까지 전 과정 학습
- **시니어 개발자** — 복잡한 AI 통합 프로젝트 워크플로우 업그레이드

## 학습 경로

### Quick Start (5분)
Snake 게임 미니 프로젝트로 AI 코딩 즉시 체험

### Stage 0-1: 기초
- 아이디어 검증
- AI IDE 도구 소개
- 프로토타입 빌딩
- AI 기능 통합
- 프로젝트 실습 + 사용자 피드백 반복

### Stage 2: 풀스택 개발
- 프론트엔드 UI 디자인
- 백엔드 API 구축
- 데이터베이스 구현
- 배포 (CloudBase, Vercel, Zeabur)
- 결제 통합 (Stripe)
- 멀티 프로덕트 아키텍처

### Stage 3: 고급 개발
- **Claude Code 워크플로우**
- 크로스 플랫폼: WeChat 미니프로그램, Android/iOS, 데스크톱
- 고급 RAG 시스템
- 특수 구현

### 부록
9개 지식 도메인에 걸친 80+ 인터랙티브 참조 주제

## 기술 스택
- 모던 프론트엔드 프레임워크
- Supabase (백엔드)
- 배포: CloudBase, Vercel, Zeabur
- 결제: Stripe
- 크로스 플랫폼: Expo, Electron, Qt
- AI 워크플로우: Dify, LangGraph
